# Naive-Bayes-Titanic-case-study
This repository presents a Machine Learning case study that predicts passenger survival on the Titanic using the Naive Bayes classification algorithm. The project follows a complete ML pipeline, including data preprocessing, exploratory data analysis (EDA), model training, and performance evaluation.

Problem Statement:
To build a classification model that predicts whether a passenger survived the Titanic disaster based on attributes such as age, gender, passenger class, fare, and other relevant features.

Dataset:

Dataset: Titanic Dataset
Type: Classification Problem
Target Variable: Survival (0 = Did Not Survive, 1 = Survived)

Steps Performed:

Data Cleaning and Handling Missing Values
Exploratory Data Analysis (EDA)
Feature Encoding (Categorical Variables)
Feature Scaling (if applied)
Train-Test Split
Model Training using Naive Bayes
Model Evaluation

Model Evaluation:

Accuracy Score
Confusion Matrix
Precision, Recall, F1-Score

Technologies Used:

Python
Pandas
NumPy
Matplotlib / Seaborn
Scikit-learn
Jupyter Notebook
